# When Crime Calls, Does Help Follow?

Final project for **02806 Social Data Analysis and Visualization** at DTU.

This project explores the relationship between violent crime and emergency medical response times in San Francisco. We use open data from DataSF, focusing on **Assault and Robbery incidents** and **Medical Incident unit responses** during the final analysis period **2016-2025**.

The website presents the main findings as a magazine-style data story with interactive maps and visualizations. The explainer notebook contains the methodology, preprocessing, analysis, visualization choices, and discussion.

## Files

- `index.html` — main data story website
- `assignmentB_group47.ipynb` — explainer notebook
- `images/` — exported interactive visualizations used in the website
